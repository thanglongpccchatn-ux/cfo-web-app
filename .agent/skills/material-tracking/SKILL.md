# Goal

Hướng dẫn nhập và quản lý danh sách vật tư theo từng hợp đồng/dự án, bao gồm
theo dõi số lượng, đơn giá, VAT, thành tiền, và trạng thái thanh toán NCC.

---

# Instructions

## Cấu trúc 1 dòng vật tư:

| Field | Mô tả | Bắt buộc |
|-------|-------|----------|
| Nhóm VTHH | Phân loại (VD: "Vật tư phụ", "Cơ điện", "Hoàn thiện") | ✅ |
| Ngày | Ngày nhập/mua hàng | ✅ |
| Nhà cung cấp | Tên NCC | ✗ |
| Tên sản phẩm | Quy cách, chủng loại | ✅ |
| ĐVT | Đơn vị tính (cái, m², kg, m...) | ✅ |
| Số lượng | Số lượng thực tế | ✅ |
| Đơn giá | Giá mỗi đơn vị (chưa VAT) | ✅ |
| VAT (%) | Thuế VAT của hóa đơn vật tư | ✗ |
| Thành tiền | **TỰ TÍNH**: Số lượng × Đơn giá × (1 + VAT%) | auto |
| Đã thanh toán | Số tiền đã trả NCC | ✗ |

## Quy trình nhập vật tư:

1. Vào hợp đồng → Tab "Vật Tư" hoặc module MaterialTracking
2. Nhấn **"THÊM DÒNG MỚI"** → hàng edit mới xuất hiện đầu bảng
3. Điền các trường (Nhóm, Ngày, Tên SP — bắt buộc)
4. Nhập Số lượng + Đơn giá + VAT → **Thành tiền tự tính**
   - Công thức: `Thành tiền = Qty × Price × (1 + VAT/100)`
5. Nhập Đã thanh toán (nếu đã trả NCC một phần)
6. Nhấn ✅ để lưu

## Xem tổng quan:
- **Tổng giá trị**: Tổng tất cả Thành tiền (gồm VAT)
- **Đã thanh toán**: Tổng paid_amount
- **Còn nợ NCC**: = Tổng giá trị − Đã thanh toán

# Examples

## Ví dụ 1: Nhập vật tư thép

**Input từ user:** "Nhập 500kg thép Ø10, đơn giá 25,000/kg, VAT 8%, NCC Thép Miền Nam, chưa thanh toán"

**Điền vào form:**
| Field | Giá trị |
|-------|---------|
| Nhóm | Vật tư kết cấu |
| Ngày | (hôm nay) |
| NCC | Thép Miền Nam |
| Tên SP | Thép tròn Ø10 |
| ĐVT | kg |
| Số lượng | 500 |
| Đơn giá | 25,000 |
| VAT | 8% |
| **Thành tiền** | **13,500,000 VNĐ** |
| Đã TT | 0 |

## Ví dụ 2: Cập nhật thanh toán NCC

**User:** "Đã trả NCC Thép Miền Nam 10 triệu cho dòng vật tư thép Ø10"

→ Nhấn Edit (✏️) dòng đó → Cập nhật `Đã thanh toán = 10,000,000` → Lưu ✅

Kết quả: Còn nợ NCC sẽ giảm xuống còn 3,500,000 VNĐ
