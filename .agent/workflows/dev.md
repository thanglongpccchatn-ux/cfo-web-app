## Development Workflow

1. Start the Vite dev server:
```bash
cd e:\AG\cfo-web-app && npm run dev
```

2. Install dependencies if needed:
```bash
cd e:\AG\cfo-web-app && npm install
```

3. Build for production:
```bash
cd e:\AG\cfo-web-app && npm run build
```

4. Git status check:
```bash
cd e:\AG\cfo-web-app && git status
```

5. Git add and commit:
```bash
cd e:\AG\cfo-web-app && git add -A && git commit -m "update"
```

6. Git push:
```bash
cd e:\AG\cfo-web-app && git push
```
